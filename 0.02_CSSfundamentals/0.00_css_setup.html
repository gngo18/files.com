CSS Notes

Purpose of CSS

  • To give the outside of the site a unique look and feel
  • Alter colors, fonts, backgrounds, etc.
  • To implemnet some effects and light animations

CSS Statements

Four main components including the follwing:

  1. Declaration:the statement as a whole with curly braces
  2. Selector:the name of the part you want to style
  3. Propertyproperties to edit (ex. width, height, background, etc.)
  4. Value:numbers, colors, urls - values that we assign
